## Flujo operativo
```
Lead llega (TikTok / IG / WhatsApp / estados / Telegram / inDrive)
    → Quien esté disponible atiende (por turno o disponibilidad)
        → Envía catálogo + cierra venta (copys/whatsapp.md)
            → Recibe comprobante de pago
                → Paga al proveedor
                    → Proveedor entrega la cuenta/perfil
                        → Se envía al cliente (activación <1h)
                            → Se registra en el CRM (marketing/remarketing-crm.md)
```
