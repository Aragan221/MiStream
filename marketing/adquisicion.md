# Estrategia de Adquisición — MiStream

> Cómo conseguir clientes. De lo orgánico (gratis) a la pauta (cuando haya ganancias).
> Para playbooks completos de embudo, ofertas y Meta Ads, ver el repo `crecio-cerebro` (`skills/marketing/`).

---

## Fase 1 — Orgánico (HOY, costo $0)

### Canal estrella: inDrive
- Ofrecer pantallas a pasajeros tras una buena carrera (ver `copys/indrive.md`).
- Ventaja: confianza cara a cara. Gancho de precio de primera vez + referidos.

### Contenido orgánico TikTok + Instagram
- Videos cortos mostrando precios, combos y el gancho "Spotify a $14.000 vs $22.000 de otros".
- Antes de publicar links: calentar las cuentas (TikTok pide ~1.000 seguidores para link en bio).
- Para guiones que enganchan, ver `crecio-cerebro/skills/contenido/guionizacion.md`.

### Referidos
- Cada cliente puede traer más clientes (ver `negocio/referidos.md`). Crecimiento exponencial sin pagar comisión en efectivo.

---

## Fase 2 — Pauta (cuando haya ganancias acumuladas)
- Meta Ads con presupuesto bajo para empezar ($5.000–$10.000/día).
- Probar 3–4 anuncios en paralelo y medir cuál convierte.
- Solo escalar lo que ya da resultado.
- Playbook completo: `crecio-cerebro/skills/marketing/meta-ads.md`.

## Embudo objetivo
```
Contenido (TikTok/IG) o inDrive
    → DM / WhatsApp
        → Copys de venta (precios + valor)
            → Pago (Nequi/Bancolombia) + comprobante
                → Activación < 1h
                    → Recordatorio de recarga mes siguiente (retención)
                        → Pedir referido (crecimiento)
```

> Para profundizar el Buyer's Journey y la oferta irresistible: `crecio-cerebro/skills/marketing/embudo.md` y `outreach-ofertas.md`.
