# Modelo de Negocio — MiStream

## Qué es
MiStream revende **perfiles privados de streaming** (Netflix, Disney+, Max, Spotify, YouTube, etc.) a clientes finales en Colombia. No producimos nada: hacemos **arbitraje**.

```
Proveedor (barato)  →  MiStream (revende con margen)  →  Cliente final
        │                        │                            │
   da el perfil            atiende, vende             paga y disfruta
   + soporte técnico       y cobra
```

## Cómo funciona (paso a paso)
1. El cliente nos contacta (inDrive, WhatsApp, IG, TikTok, Telegram).
2. Le mostramos precios / combos y, sobre todo, **el valor** (ver `propuesta-valor/`).
3. **Paga primero** (Nequi / Bancolombia) y envía comprobante.
4. Le compramos el perfil al proveedor y **activamos en menos de 1 hora**.
5. Si el perfil falla, el **proveedor da el soporte**; nosotros gestionamos la reposición con el cliente.

## Roles
| Rol | Quién | Qué hace |
|-----|-------|----------|
| Proveedor | Externo | Entrega perfiles + soporte técnico |
| Operador / ventas | Juan Manuel + parceros | Atiende, vende, cobra, activa |
| Cliente | Final | Paga y usa la pantalla |

## Por qué este proyecto existe
- **Ingreso por arbitraje** con márgenes sanos ($6.000–$8.500 por pantalla).
- **Validar** que el modelo y los conocimientos de marketing/ventas (del repo `crecio-cerebro`) aplican a un negocio de producto digital, no solo a servicios profesionales.
- Construir un **activo escalable**: hoy 1 a 1 por WhatsApp → mañana landing + Telegram + sistema de referidos.
